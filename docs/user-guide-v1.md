# H&NTrip — guia rápido

1. Crie uma viagem e informe datas, destino, moeda e orçamento.
2. Adicione participantes informativos; o acesso ao sistema é gerenciado separadamente em Configurações.
3. Monte o roteiro, registre gastos e acompanhe o checklist.
4. Salve locais e referências de reserva sem cadastrar senhas ou cartões.
5. Envie documentos e fotos; os arquivos são privados e os downloads usam acesso temporário.
6. Escolha fotos favoritas e uma capa. Legenda, data e local podem ser editados.
7. Consulte álbum, estatísticas, alertas e histórico.
8. Use Arquivados para restaurar viagens, fotos, documentos e locais.

Owner gerencia membros e convites. Admin trabalha no conteúdo, mas não altera acessos. Ao sair de um aparelho compartilhado, use o botão Sair.
